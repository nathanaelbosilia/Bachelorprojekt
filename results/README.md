# Python version
The python version used for development and testing is 3.12.2.

# Change directory to reporting
``cd reporting``

# Create adn activate venv
``python -m venv venv``
``source venv/bin/activate``

# Install dependencies
``pip install -r requirements.txt``

# Run the script
`` python main.py``

# Deactivate venv
``deactivate``
